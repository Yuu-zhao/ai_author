# 多页面应用说明

## 页面结构

本项目使用 Streamlit 的多页面应用功能，每个功能模块都是独立的页面：

- **首页** (`ui.py`) - 小说创作工作台，包含章节生成和快速导航
- **1_基本信息.py** - 基本信息管理（书名、简介、标签）
- **2_剧情大纲.py** - 剧情大纲管理
- **3_设定管理.py** - 设定管理（世界观、修炼体系等）
- **4_角色管理.py** - 角色管理
- **5_剧情状态.py** - 剧情状态管理
- **6_章节管理.py** - 章节管理（章节目录和分卷管理）

## 使用方式

### 1. 从首页跳转

在首页点击对应的功能按钮，即可跳转到相应页面。

### 2. 直接访问

Streamlit 会自动在侧边栏显示所有页面，可以直接点击侧边栏的页面名称进行跳转。

### 3. 返回首页

每个功能页面都有"🔙 返回首页"按钮，点击即可返回首页。

### 4. 浏览器导航

- 支持浏览器的前进/后退按钮
- 每个页面都有独立的 URL
- 可以收藏特定页面

## 页面命名规则

Streamlit 会根据文件名自动排序和显示：
- 数字前缀控制显示顺序（1_、2_、3_...）
- 文件名会显示在侧边栏导航中

## 优势

1. **模块化**：每个功能独立，便于维护
2. **导航清晰**：侧边栏自动显示所有页面
3. **URL 支持**：每个页面有独立 URL
4. **浏览器支持**：支持前进/后退按钮
5. **可分享**：可以分享特定页面的链接
