# 🚀 启动指南

## 快速启动（3 步）

### 1️⃣ 安装依赖

```bash
pip install -r requirements.txt
```

或者使用虚拟环境（推荐）：

```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
# macOS/Linux:
source venv/bin/activate
# Windows:
# venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 2️⃣ 配置 API Key

创建 `.env` 文件（在项目根目录）：

```bash
# 方式一：使用命令行创建
echo "OPENAI_API_KEY=your_api_key_here" > .env

# 方式二：手动创建文件，内容如下：
# OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**重要：** 将 `your_api_key_here` 替换为你的实际 OpenAI API Key

### 3️⃣ 启动界面

```bash
streamlit run ui.py
```

启动后，浏览器会自动打开界面（通常是 `http://localhost:8501`）

---

## 常见问题

### Q: 提示 "ModuleNotFoundError: No module named 'streamlit'"
**A:** 需要先安装依赖：`pip install -r requirements.txt`

### Q: 提示 "OPENAI_API_KEY not found"
**A:** 检查 `.env` 文件是否存在，且 API Key 配置正确

### Q: 浏览器没有自动打开
**A:** 手动访问终端中显示的 URL（通常是 `http://localhost:8501`）

### Q: 如何停止界面
**A:** 在终端按 `Ctrl + C`

## 验证安装

运行以下命令检查依赖是否安装成功：

```bash
python3 -c "import streamlit; import openai; from dotenv import load_dotenv; print('✅ 所有依赖已安装')"
```

如果看到 ✅ 提示，说明安装成功！
